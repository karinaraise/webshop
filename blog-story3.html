#Blog 3

Les mijn ervaringen tijdens het maken van deze webshop.

Google Ads en landingspagina

Landingspage
Het doel van mijn advertentie is zorgen dat nieuwe gebruikers weten hoe ze moeten bestellen en direct hun eerst aankoop doen. Op die pagina leg ik daarom uit waarom ze moeten bestellen en hoe zij kunnen bestellen. Om hun alvast inspiratie en idee te geven uit welke producten zij kunnen kiezen, heb ik een foto slider gemaakt en kunnen zij de categorieën zien. En om het vertrouw te winnen van de gebruikers, laat ik zien wat andere klanten zeggen.

Google Ads stappen
1. Als eerste heb ik bepaald een doel voor mijn campagne. Ik heb voor Verkopen gekozen om te zorgen dat bezoekers mijn producten gaan kopen.

hover this image to zoom

2. Als type campagne heb ik gekozen voor Zoeken. Omdat mensen die specifiek naar een geïllustreerd cadeau zoeken op mijn site kunnen komen.

hover this image to zoom


3. De resultaten worden behaald wanneer gebruikers op mijn site komen. De link van de landingspagina heb ik daarin gezet. Op de screenshot is de oude link te zien, deze ik later aangepast naar: https://st425001.cmd22a.cmi.hanze.nl/webshop/illustratie.html. En mijn eerste campagne heet: eerste kopers vinden.

hover this image to zoom


4. Ik wil het beste uit deze campagne eruit halen en daarom heb ik de optie “klikken” gekozen om de strategie te maximaliseren.

5. Ik heb bij netwerken niks aangevinkt om te voorkomen dat ik extra kosten krijg.

hover this image to zoom


6. Bij locaties en talen heb ik wereldwijd gekozen omdat ik ook digital product aanbied. Voor de taal, voor deze campagne heb ik Nederlands gekozen omdat de inhoud van mijn website Nederlands geschreven is en de producten alleen naar Nederland en België verzonden worden.

hover this image to zoom


7. Om de juiste woorden op geven heb ik de tool Zoekwoordplanner gebruikt. Ik heb de woorden gekozen die het meest worden gebruikt. Op deze manier wordt mijn site sneller gevonden en heb ik meer kans op verkoop.

hover this image to zoom


8. Deze stap is belangrijk omdat dat is wat potentiële klanten gaan zien. Ik heb de woorden van de vorige stap hierin verwerkt om zo optimaal de resultaten te halen die ik wil. De woorden zijn verwerkt in de kopen en de beschrijving.

hover this image to zoom


9. Daarna heb ik extra sitelinks toegevoegd om de bezoeker de mogelijkheid te bieden direct op een product te komen. Ook heb ik email en telefoon gegevens toegevoegd. Deze extensies zorgen ervoor dat mijn advertentie langer wordt en meer op valt. Ook worden de aantal klikfrenquentie ermee verhoogd.

hover this image to zoom


10. Laatste stap is alles controleren. Wanneer dit goed zou zijn, kan ik het publiceren. Ik heb per ongeluk op publiceren geklikt maar ik heb direct kunnen verwijderen voor dat het mij echt geld zou kosten.

hover this image to zoom


Superleuk en interessant om dit te doen. Het is wel een vak apart om de juiste woorden weten toe te passen.


Karina da Silva, Roden 15-06-2022
